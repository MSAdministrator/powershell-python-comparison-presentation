# Python Dictionary's Continued

## Get a count of key/value pairs
```python
len(my_dict)      # 3
```
## Access a value by referencing the key name
```python
my_dict['Shape']    # Square
```
## Adding to a dictionary
```python
my_dict.update({
    'Time': 'Now'
})

a = 'Time'
my_dict.update(a='Now')
```
